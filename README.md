# 🎓 QR-ОбучAI

Система регистрации инкассаторов на обучение с автоматической генерацией отчётных документов.

## 📋 Описание

Проект мигрирован на FastAPI + Vercel для лучшей масштабируемости и контроля.

### Основные функции:

- ✅ **Регистрация работников** через QR-код с геолокацией
- ✅ **Проверка дубликатов** в пределах квартала
- ✅ **Автоматическое заполнение** отчётных документов (Google Sheets/Docs)
- ✅ **Уведомления в Telegram** ответственным сотрудникам
- ✅ **Ежедневные отчёты** о регистрациях
- ✅ **Архивация данных** по месяцам
- ✅ **Панель управления** для всех ГОСБ
